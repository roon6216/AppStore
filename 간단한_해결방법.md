# Google 로그인 "인증 안됐다" 해결 방법 (2분 소요)

## 문제
"인증되지 않은 앱" 또는 "인증 안됐다" 메시지가 나타나는 경우

## 해결 방법 (매우 간단!)

### 1단계: OAuth 동의 화면 설정 (1분)

1. https://console.cloud.google.com/ 접속
2. 같은 프로젝트 선택
3. 왼쪽 메뉴에서 **API 및 서비스** > **OAuth 동의 화면** 클릭
4. **사용자 유형 선택**:
   - **외부** 선택 (가장 간단함)
   - **만들기** 클릭
5. **앱 정보** 입력:
   - **앱 이름**: 출장 체크리스트 (아무거나)
   - **사용자 지원 이메일**: 본인 이메일 선택
   - **앱 로고**: 건너뛰기 (선택사항)
   - **앱 도메인**: 건너뛰기
   - **개발자 연락처 정보**: 본인 이메일 입력
6. **저장 후 계속** 클릭
7. **범위** 화면에서 **저장 후 계속** 클릭 (기본값 그대로)
8. **테스트 사용자** 화면에서:
   - **+ ADD USERS** 클릭
   - 사용할 Google 이메일 주소 입력 (본인, 동생 등)
   - **추가** 클릭
9. **저장 후 계속** 클릭
10. **요약** 화면에서 **대시보드로 돌아가기** 클릭

### 2단계: 완료!

이제 테스트 사용자로 추가한 이메일로 로그인하면 정상 작동합니다!

## 참고
- 테스트 사용자로 추가하지 않으면 "인증되지 않은 앱" 경고가 나타날 수 있습니다
- 나중에 앱을 공개하려면 Google 검토를 받아야 하지만, 지금은 테스트 사용자만 추가하면 됩니다
