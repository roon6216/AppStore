# Google Sheets API 활성화 방법 (1분 소요)

## 에러 해결 방법

에러 메시지에 나온 링크를 클릭하거나 아래 단계를 따라하세요:

### 방법 1: 에러 메시지의 링크 사용 (가장 빠름)

에러 메시지에 나온 링크를 클릭:
```
https://console.developers.google.com/apis/api/sheets.googleapis.com/overview?project=693056998078
```

**사용** 버튼 클릭 → 완료!

### 방법 2: 직접 활성화

1. https://console.cloud.google.com/ 접속
2. 프로젝트 선택 (프로젝트 ID: 693056998078)
3. 왼쪽 메뉴: **API 및 서비스** > **라이브러리** 클릭
4. 검색창에 "Google Sheets API" 입력
5. **Google Sheets API** 클릭
6. **사용** 버튼 클릭
7. 완료!

## 활성화 후

- 몇 분 정도 기다린 후 앱을 다시 실행하세요
- API 활성화가 전파되는데 시간이 걸릴 수 있습니다
- 그래도 안 되면 앱을 완전히 종료하고 다시 실행하세요

## 확인 방법

활성화가 완료되면:
- "주행거리 읽기 실패" 에러가 사라집니다
- 스프레드시트에서 데이터를 읽고 쓸 수 있습니다
